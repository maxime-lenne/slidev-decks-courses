### Prérequis & Objectifs

::left::

#### Prérequis

- Bases **LLM** et **tool calling** (function calling)
- **Python** intermédiaire (décorateurs, type hints, async)
- Notions de **JSON-RPC** ou de protocole client/serveur
- Avoir déjà utilisé un MCP côté client (Claude Desktop, VS Code) — souhaitable

**Niveau :** mixte technique · juniors → seniors familiers agents

::right::

#### Objectifs

À l'issue de cette session vous serez capable de :

- Décrire l'**architecture** host / client / server de MCP
- Lire un **échange JSON-RPC** (initialize, capabilities, tools/call)
- Distinguer les primitives **server** (tools, resources, prompts) des primitives **client** (sampling, elicitation, roots)
- Construire un **premier serveur** MCP avec **FastMCP** et l'inspecter
- Savoir qu'un MCP **suit les règles SWE** (tests, CI/CD, registry, observabilité)
