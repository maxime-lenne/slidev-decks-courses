### Copilot, là où vous travaillez déjà

<br>

<div class="grid grid-cols-2 gap-8 mt-4 text-sm">

<div>

#### Dans la suite Microsoft 365

- 📝 **Word** — rédiger, résumer, reformuler un doc
- 📊 **Excel** — analyser, formules, tendances, graphiques
- 📧 **Outlook** — trier, résumer un fil, rédiger une réponse
- 💬 **Teams** — notes de réunion, résumé, actions à suivre
- 📽️ **PowerPoint** — générer une slide depuis un texte
- 🔎 **Copilot Chat** — question sur tous vos fichiers M365

</div>

<div>

#### Le principe : le *grounding*

Copilot voit (avec vos droits) vos **mails, documents, réunions** via Microsoft Graph.

- Réponses **ancrées sur vos données** réelles
- Respecte les **permissions** existantes
- Ne montre que ce à quoi **vous avez déjà accès**

<div class="text-xs opacity-70 mt-3">C'est la grande différence avec un chat « nu » : Copilot connaît <strong>votre</strong> contexte.</div>

</div>

</div>

### ChatGPT vs Copilot : complémentaires

<br>

<div class="grid grid-cols-2 gap-8 mt-4 text-sm">

<div class="border-l-4 border-[#10a37f] pl-4">

#### Réflexe ChatGPT

- Tâche **ouverte**, créative, exploratoire
- Contenu **hors données d'entreprise**
- Prototyper un GPT / un mini-outil
- Travailler sur un texte **que vous collez**

<div class="text-xs opacity-70 mt-3">« J'ouvre un chat pour réfléchir / produire. »</div>

</div>

<div class="border-l-4 border-[#0078d4] pl-4">

#### Réflexe Copilot

- Tâche **sur vos fichiers / mails M365**
- Besoin d'**ancrage** sur les données réelles
- **Confidentialité** entreprise requise
- Rester **dans le flux** de travail (Word, Teams…)

<div class="text-xs opacity-70 mt-3">« Je reste dans mes outils, l'IA connaît mon contexte. »</div>

</div>

</div>

<div class="text-center text-sm mt-6 opacity-70">
Beaucoup de pros utilisent <strong>les deux</strong> — le bon réflexe, c'est de savoir lequel dégainer.
</div>
